# Wordle

**游戏玩法：**  
在原版的《Wordle》中，玩家要在一天内用六次机会内猜中某个有五字英文字母的词汇。每次尝试后，玩家可能得到三种反馈：绿色表示字母位置正确；黄色表示答案包含该字母但位置错误；灰色表示答案没有该字母。  
本版本有一些简化，包括使用不同的标记展示正确与错误，词汇库可能与原版词汇库有所不同等等。
